## 色彩系统

Hopes 博客的主题遵循“温暖、阅读优先”的原则。我们采用 HSL 变量统一管理颜色，让 Tailwind 原子类自动映射到设计稿中的 token。

```css
:root {
  --background: 32 60% 96%;
  --foreground: 26 37% 18%;
}
```

通过 CSS 自定义属性，可以在暗色模式下覆写同一组变量，而无需成倍编写样式。

## 深色模式的切换

Tailwind 4 允许我们声明 `darkMode: 'class'`，然后在 React 客户端组件中监听系统主题或按钮交互来切换 `document.documentElement.classList`。

## 可访问性

- 保持文本与背景的对比度大于 4.5。
- 为所有交互元素提供 `focus-visible` 样式。
- 使用语义化的 HTML 结构，搭配 `aria` 属性补充额外信息。

## 动效与体验

微动效应当轻量且有意义。例如在按钮悬停时增加 `ring`，提示当前的焦点位置，同时通过 `prefers-reduced-motion` 降级动画。

## 总结

设计语言不是一次性工作，需要结合用户反馈持续迭代。保持 token 化的心智模型有助于快速调整整体风格。
