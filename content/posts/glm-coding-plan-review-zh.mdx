## 配置使用体验

### 配置过程

配置GLM Coding Plan相对简单，主要步骤如下：

1. **注册账号**：在智谱开放平台注册账号
2. **订阅套餐**：选择合适的编程套餐
3. **获取API Key**：在个人中心创建API Key
4. **配置工具**：在编程工具中配置API

以Claude Code为例，配置方式：

```bash
# 设置环境变量
export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/anthropic"
export ANTHROPIC_AUTH_TOKEN="your-api-key"
```
